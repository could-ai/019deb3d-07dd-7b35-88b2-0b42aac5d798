# System Core Terminal

System Core Terminalは、仮想的なシステム制御や能力管理をシミュレートするFlutterアプリケーションです。SF風のターミナルインターフェースを通じて、システムの起動、能力の登録、およびAI的モデルとの対話を行うことができます。

## 主な機能

- **スプラッシュスクリーン**: アプリ起動時のシステム初期化アニメーション。
- **システム初期化**: `Core_Init` プロセスによる、アイデンティティ、メモリ、意思決定エンジン、能力管理システムの初期セットアップ。
- **能力登録フロー**: ユーザーから初期能力データ（能力名、定義）を受け取り、システムに登録。
- **ステータス管理**: 「待機中」「初期化中」「能力入力待機」「確定処理中」「動作中」など、システムの状態をリアルタイムにUIへ反映。
- **インタラクションループ**: 起動後の対話型ターミナル。ユーザーの入力に対して「意思決定エンジン（Decision Engine）」が評価し、仮想的な応答や音声出力をシミュレート。

## 技術スタック

- Flutter
- Dart
- 状態管理: `StatefulWidget` を用いた状態遷移の制御

## セットアップと実行

1. Flutter SDKがインストールされていることを確認します。
2. リポジトリをクローンまたはダウンロードします。
3. プロジェクトディレクトリで以下のコマンドを実行し、依存関係を取得します：
   ```bash
   flutter pub get
   ```
4. アプリケーションを実行します：
   ```bash
   flutter run
   ```

## アプリケーションの構成

- `lib/main.dart` - アプリケーションのエントリーポイント
- `lib/splash_screen.dart` - 起動時のスプラッシュ画面
- `lib/main_screen.dart` - メインUI・ステータス管理およびインタラクション画面
- `lib/core_system.dart` - システムのロジック・モデル定義（`CoreSystem`, `AbilityManager`, `DecisionEngine`など）
